# SOTN Shuffler

A tool for modifying various things about SOTN, including the locations of rooms, in the form of change files. Intended to be used in conjunction with the [BIN Patcher](https://github.com/SestrenExsis/BIN-Patcher) tool, which turns those JSON files into patch (PPF) files.

## Current status of the project

This project is still actively in the prototyping phase of development. It almost definitely has bugs, will cause softlocks, and may even corrupt the game BIN in its current state. Use at your own risk.
